How it works

  1. Click Sign in with Blink. A browser tab opens to Blink’s login page.
  2. Complete the Blink sign-in and MFA steps. When finished, you’ll see a success message.
  3. Return to this screen; the status updates to Signed in once tokens are stored.

If the flow stalls, press Reset link to clear stored tokens and try again.
