Credentials

Sign in with your Blink account to fetch and refresh tokens. You’ll be prompted for the 2FA PIN after the first login attempt.

After you start login, enter the 6-digit code Blink sends to your phone.

Need a refresher?

You can still extract an access_token and refresh_token from the Blink app or API and paste them here, then press Save Tokens.

If you provide your Blink email and password, the Login & Fetch Tokens button will request fresh credentials automatically. The PIN field appears after Blink sends a 2FA challenge.

The Refresh Tokens button requests a fresh pair from Blink using the stored refresh token so your Homebridge instance stays authorised.
