Need a refresher?

Use a proxy or device inspection tool to obtain your Blink access_token and refresh_token directly from the official app. Paste them here, then press Save Tokens.

The Refresh Tokens button requests a fresh pair from Blink using the stored refresh token so your Homebridge instance stays authorised.
